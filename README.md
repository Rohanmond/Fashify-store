## [FashiFy Store](https://fashify.netlify.app/)

### Introduction -
FashiFy Store is an Indian Fashion ecommerce company.


### Features - 
- Product Management. 
- Cart Management
- Wishlist Management
- Authentication
- profile dropdown
- Responsive Website

### Tech Stack - 
- ReactJs, Vanilla CSS integrated with [GlexUI](https://glex-ui.vercel.app/) Component Library elements.
